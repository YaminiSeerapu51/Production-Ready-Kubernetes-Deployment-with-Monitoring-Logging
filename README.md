# Production-Ready-Kubernetes-Deployment-with-Monitoring-Logging
- Deployed a microservice application on Kubernetes with Deployments, Services, ConfigMaps, Secrets, HPA.
- Implemented Prometheus + Grafana for real-time monitoring, custom metrics, dashboards, and alert rules.
- Configured ELK / EFK logging stack for central log aggregation and faster debugging.
- Performed root-cause analysis for CrashLoopBackOff, OOMKilled, high CPU, failed liveness/readiness probes.
- Used Kubernetes NetworkPolicies and RBAC for least-privilege security.
- Integrated app deployment with CI/CD pipeline using Kubernetes manifests / Helm chart
